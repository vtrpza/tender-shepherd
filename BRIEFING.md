#### 6. Terminal Animado

O terminal é a peça que vende. Simula o agente de tráfego rodando uma auditoria ao vivo.

**Design do terminal:**

- Barra superior com 3 dots macOS (vermelho, amarelo, verde)
- Título na barra: `empresa-autonoma — agente de tráfego`
- Background ligeiramente mais claro que o body (`#111` range)
- Border 1px sutil, `border-radius` 12–16px
- `box-shadow` pesado: `0 24px 80px rgba(0,0,0,0.5)`

**Roteiro EXATO das linhas** (cada uma aparece com delay de ~900ms):

```
$ claude -p "Auditar campanhas Meta Ads dos últimos 7 dias"
⠋ Conectando ao Meta Ads via MCP...
✓ 14 campanhas analisadas em 23 segundos
⚠ Campanha "Remarketing_SP": CPA 340% acima do target
⚠ 47 keywords sem conversão queimando R$214/dia
✓ Keywords negativas adicionadas automaticamente
✓ Relatório gerado → /reports/audit-mar-2026.pdf
💰 Economia identificada: R$12.847/mês
```

Cores semânticas:
- Prompt `$` → accent
- Output geral → muted
- `✓` → verde (sucesso)
- `⚠` → vermelho/amber (alerta)
- `💰` linha final → amber bold (clímax — cursor piscando aqui)

O terminal deve ter animação typewriter ou fade-in-up linha a linha. Depois da última linha, o cursor fica piscando indefinidamente.

#### 7. Scroll hint

Seta animada apontando pra baixo, label "veja como funciona".

### Seção 9 — FAQ (Matando objeções)

**Headline:** "Mas e se..." Já respondi.

**Formato:** Cards em grid 2×3 (ou accordion no mobile).

**P: Preciso saber programar?**

Não. Zero código. Os scripts vêm prontos — você copia, cola, ajusta 3 variáveis (nome da empresa, suas regras, seus canais) e roda. Se você sabe instalar um app no celular, consegue.

**P: Quanto custa manter o Claude rodando?**

O plano Pro do Claude custa ~R$115/mês. Cada agente noturno consome centavos em tokens. Custo total: menos de R$150/mês para 3 agentes 24/7. Compare com R$7.500 em equipe operacional.

**P: E se a IA fizer besteira?**

Você define as regras. O agente só faz o que você autorizar. Quer que ele SUGIRA pausar uma campanha sem executar? Configurável. Quer que ele mande confirmação no WhatsApp antes de agir? Também. Você tem controle granular sobre cada ação. Eu mostro como.

**P: Isso substitui minha equipe?**

Não — e esse não é o objetivo. O agente cuida dos 80% repetitivos e operacionais. Sua equipe fica livre pros 20% que precisam de inteligência humana: estratégia, criatividade, relacionamento. Você não demite. Você libera.

**P: Já vi muito guru vendendo milagre com IA.**

Eu também. Por isso não prometo "6 em 7" nem "renda passiva". O que prometo: scripts que funcionam, demonstrações reais, e um passo a passo honesto de engenheiro. Não gostou? 7 dias pra pedir reembolso. Sem formulário.

**P: Por que tão barato?**

### 7.3 Componente Terminal

O `Terminal.astro` recebe props para ser reutilizável nas 3 ocorrências (hero + 2 na seção de prova técnica):

```typescript
interface Props {
  title: string;       // texto na barra superior
  lines: TerminalLine[]; // roteiro de linhas
  autoplay?: boolean;  // true = anima no load, false = anima no viewport
}

interface TerminalLine {
  text: string;
  color?: 'default' | 'accent' | 'green' | 'red' | 'amber' | 'muted';

}
```

## 9. Notas Finais

### O que faz essa LP vender

1. **O terminal animado é o herói.** Ele PROVA sem argumentar. Gaste tempo nele.
2. **O Vitor Pouza é o diferencial.** Não é um anônimo vendendo curso. É um engenheiro de 10+ anos que já cobra R$5k–30k por isso como serviço. A seção "Quem ensina" é a segunda mais importante depois do terminal.
3. **Os números em R$ são âncoras.** R$7.500/mês → R$90.000/ano → R$197. O cérebro faz a conta.
4. **A escassez é real.** Contador de vagas = gatilho emocional de "vou perder o preço".
5. **O tom anti-guru é o diferencial de branding.** O empresário brasileiro está FARTO. Cada frase da copy precisa soar como alguém que sabe do que fala, não como alguém que quer vender.

### O que NÃO fazer

- NÃO inventar depoimentos/testimonials (é pré-venda, fabricar é antiético)
- NÃO usar stock photos ou ilustrações genéricas (o terminal é o visual)
- NÃO criar popup de saída
- NÃO usar countdown timer falso
- NÃO usar chat widget
- NÃO mencionar "ChatGPT" em nenhum momento
- NÃO mudar o tom de voz — se não soar como o Vitor Pouza escreveria, reescreva
- NÃO usar emojis em excesso (máximo: os usados nos badges e no terminal)
- NÃO colocar "por Vitor Pouza" no hero — a seção "Quem ensina" cuida da autoridade no timing certo
